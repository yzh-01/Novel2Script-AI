# Novel2Script AI — Demo 演示脚本

**时长**：约 3-4 分钟  
**录制工具**：OBS Studio（免费）/ Loom / 系统自带录屏  
**配音**：用手机录音后合成，或录屏时同步讲述

---

## 开头（15s）

| 画面 | 旁白 |
|------|------|
| 打开 `https://novel2-script-ai.vercel.app/`，展示首页 | "大家好，这是 Novel2Script AI——一个将小说自动转换为专业剧本格式的工具。支持电影、电视剧、短剧三种格式。" |

---

## 方式一：手动输入（40s）

| 画面 | 旁白 |
|------|------|
| 标题栏输入 "斗破苍穹" | "首先演示手动输入方式。输入剧本标题。" |
| 类型选"奇幻"，格式选"电视剧" | "选择小说类型和目标格式。这里我们选奇幻和电视剧。" |
| 在第 1 章标题输入 "陨落的天才"，内容区粘贴第一章文本 | "粘贴小说章节内容。每章包含标题和正文。" |
| 点击"添加章节"，输入第 2 章 | "可以添加多个章节，点击加号即可。" |
| 也可快速点击示例 | "也可以点击示例按钮快速填充测试数据。" |
| 点击"转换为剧本" | "点击转换为剧本，系统会调用 AI 进行解析。" |

| 画面 | 旁白 |
|------|------|

## 编辑器功能（50s）

| 画面 | 旁白 |
|------|------|
| 展示左栏 YAML 源码，右栏剧本预览 | "编辑器采用分栏设计。左侧是 YAML 源码，可直接编辑；右侧是电影剧本式预览。" |
| 切换到"YAML"视图 | "也可以只看 YAML 源码。" |
| 切换到"预览"视图 | "或者只看剧本预览，降低非程序员使用门槛。" |
| 滚动展示角色表、场景列表 | "预览包含完整角色表和逐场景的剧本内容。" |
| 修改 YAML 中某句对白 | "修改 YAML 源码，预览会实时同步。注意顶栏出现「已编辑」标识。" |
| 点击"保存到历史" | "编辑后点击保存到历史记录，修改后的版本会被保存。" |
| 按钮变为"✓ 已保存" | "保存成功。" |
| 点击"下载 YAML" | "也可以下载 YAML 文件到本地。" |

|------|------|

## 录制建议

1. **分辨率**：1920×1080（或浏览器窗口 1280×720）
2. **缩放**：浏览器缩放到 100%-125%，确保文字清晰
3. **鼠标**：操作时鼠标移动稍慢，点击前稍停顿
4. **音频**：用手机录音后再用剪映/Premiere 合成到视频
5. **剪辑**：转换过程如果较长，可加速或剪辑过渡
